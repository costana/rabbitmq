# Subindo container RabbitMQ no docker-compose

## Setup
```
git clone https://github.com/costana/rabbitmq.git
cd rabbitmq
docker-compose up

cd examples/Receive
dotnet run

cd examples/Send
dotnet run

```

## Play
Acesse http://localhost:15672
